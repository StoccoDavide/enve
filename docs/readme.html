The ENVE library¶

A library for tire and road geometrical description, handling and real-time contact evaluation

Usage¶

  1. Load .rdf file.

    enve::rdf::mesh road(
  std::string "./file.rdf" // Path to the *.rdf file
);

  2. Initialize the tire shell model.

    roadrunner::shell tire_shell(
  acme::real radius_x,   // [m]
  acme::real degree_x,   // [-]
  acme::real radius_y,   // [m]
  acme::real degree_y,   // [-]
  acme::int  ribs_number // [-]
);

    Notice that `radius_x = tire_etrto.unloadedRadius()`.

  3. Contact evaluation with rdf mesh.

    bool out = tire_shell.setup(
  enve::ground::mesh road,     // Rdf mesh
  acme::affine       affine,   // 4x4 total transformation matrix
  acme::int          threshold // Triangle evaluation number threshold
);

    Otherwise perform contact evaluation with local plane. Initialize plane object:

    acme::plane local_plane(
  acme::vec3 plane_point, // Local plane point
  acme::vec3 plane_normal // Local plane normal
);

    Perform intersection:

    bool out = tire_shell.setup(
  acme::plane  local_plane, // Local plane object
  acme::real   friction,    // Local plane friction coefficient
  acme::affine affine       // 4x4 total transformation matrix
);
  1. Data extraction:

    Variables initialization (real numbers):

    acme::vec3 contact_point;     // [m]
acme::vec3 contact_normal;    // [-]
acme::real contact_friction;  // [-]
acme::real contact_depth;     // [m]
acme::real contact_depth_dot; // [m/s]
acme::real contact_depth_old; // [m] to be initialized
acme::real time_step;         // [s] to be initialized
acme::real relative_camber;   // [rad]

    Data extraction (real numbers):

    tire_shell.contactPoint(contact_point);
tire_shell.contactNormal(contact_normal);
tire_shell.contactFriction(contact_friction);
tire_shell.contactDepth(contact_depth,
                         contact_depth_dot,
                         contact_depth_old,
                         time_step);
tire_shell.relativeCamber(relative_camber);

    Variable initialization (vectors):

    acme::int  size = tire_shell.size();
acme::vec3 contact_point_vec(size);     // [m]
acme::vec3 contact_normal_vec(size);    // [-]
acme::vecN contact_friction_vec(size);  // [-]
acme::vecN contact_depth_vec(size);     // [m]
acme::vecN contact_depth_dot_vec(size); // [m/s]
acme::real contact_depth_old_vec;       // [m] to be initialized
acme::real time_step;                   // [s] to be initialized
acme::vecN relative_camber_vec(size);   // [rad]

    Data extraction (vectors):

    tire_shell.contactPoint(contact_point_vec);
tire_shell.contactNormal(contact_normal_vec);
tire_shell.contactFriction(contact_friction_vec);
tire_shell.contactDepth(contact_depth_vec,
                         contact_depth_dot_vec,
                         contact_depth_old_vec,
                         time_step);
tire_shell.relativeCamber(relative_camber_vec);

    For more advanced function buid and read the Doxygen documentation.

Online documentation¶

Available at: https://stoccodavide.github.io/enve

Maintenance and compatibility¶

Library is being tested on Linux and MacOS, it will be soon tested also on Windows. If you have questions regarding the library or you want to report a bug, please open an issue at GitHub. If your request would contain confidential information, please send us an email.
